# 🍔 Art Burger | Landing Page

Landing page desenvolvida para uma hamburgueria fictícia com foco em apresentação visual, organização de conteúdo e responsividade.

O projeto foi criado durante o início da graduação como prática de desenvolvimento front-end utilizando apenas HTML e CSS.

---

## 🚀 Funcionalidades

* Estrutura de landing page moderna
* Sessão de apresentação da hamburgueria
* Exibição de produtos
* Layout responsivo
* Organização visual focada em experiência do usuário

---

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
